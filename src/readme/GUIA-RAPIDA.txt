🚀 GUÍA RÁPIDA - EMPEZAR AHORA
========================================

El sistema está 100% listo. Solo 3 pasos para funcionar:

🔐 PASO 1: Usuarios en Firebase Auth (5 minutos)
================================================

1. Abre: https://console.firebase.google.com
2. Selecciona: voz-arg-2c418
3. Vai a: Authentication (Autenticación)
4. Crea estos usuarios:

   Usuario 1:
   Email: admin@club.com
   Contraseña: admin@123
   
   Usuario 2:
   Email: tesorero@club.com
   Contraseña: tesorero@123

💾 PASO 2: Guardar UIDs + Crear Colección (3 minutos)
====================================================

Después de crear los usuarios:

1. Copia el UID de admin@club.com (aparece en la lista)
2. Vai a: Firestore Database
3. Crea colección: "users"
4. Agrega documento con ID = UID de admin:

{
  "nombre": "Santiago Admin",
  "email": "admin@club.com",
  "rol": "admin"
}

5. Haz lo mismo para tesorero (otra colección o documento)

📱 PASO 3: Crear Jugadores (2 minutos)
====================================

1. En Firestore, crea colección: "jugadores"
2. Agrega un documento (ID: jug_123):

{
  "nombre": "Lionel Messi",
  "dni": 12345678,
  "categoria": "A",
  "telefono": "5491123456789"
}

3. Agrega 2-3 jugadores más

Errores Comunes
===============

❌ "No puedo entrar"
   → Revisa que creaste el usuario en Auth
   → Usa: admin@club.com / admin@123

❌ "No veo deudores"
   → Verifica que los jugadores están en "jugadores" collection

❌ "WhatsApp no funciona"
   → El teléfono debe tener formato: 5491234567890 (sin espacios)

❌ "No se descarga Excel"
   → Debe haber al menos 1 pago registrado

💡 Tips
======

1. Los gráficos se actualizan automáticamente
2. Los deudores son automáticos (sin pago del mes)
3. El historial busca por DNI exacto
4. Los roles se detectan por la colección "users"
