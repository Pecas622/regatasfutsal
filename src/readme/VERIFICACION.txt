⚠️ ERRORES COMUNES Y SOLUCIONES

Error: "No puedo entrar"
□ Verifica colección "users" existe
□ Verifica que el documento tiene el campo "rol"
□ Verifica Firebase Auth tiene el usuario

Error: "Tabla de pagos vacía"
□ Registra un pago de prueba primero
□ Verifica que "pagos" colección existe

Error: "Gráfico no aparece"
□ Necesita al menos 1 pago registrado
□ Recarga la página

Error: "WhatsApp no funciona"
□ Verifica formato teléfono: 5491234567890
□ Sin espacios, sin guiones

Error: "Excel no descarga"
□ Bloquea popup
□ O revisa que tiene pagos

Error: "console error: FIREBASE NO CARGADO"
□ Revisa conexión internet
□ Verifica que firebase.js existe
□ Revisa que importa en admin-system.js
